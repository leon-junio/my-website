Saúde+ Patch Notes

Versão 2.6.0

17/09/2024

Novidades e atualizações
  • Novo sistema de segurança de login baseado em CAPTCHAS
  • Adição de CAPTCHAS aritméticos, em movimento e de texto
  • Sistema dinâmico de dificuldade para os captchas
  • Correção do sistema de controle de erros de acesso
  • Melhoria do sistema de acesso via tela de login
  • Procurar produtos por ID nas solicitações do almoxarifado
  • Correção de permissões para acesso aos IDs dos produtos de solicitadores
  • Sistema de não exclusão de retiradas quando ocorre remoções de produtos
  • Remoção de produtos de retiradas agora deixa registro permanente (quantidade inicial e alteração)
  • Devolução de itens agora tem feedback visual na guia de itens de retirada
  • Melhorias técnicas e de UI nas retiradas
  • Resolução de feedbacks de agosto de 2024
  • Correção do instalador do sistema (abrir app após a instalação)
  • Correção de biblioteca depreciada na api do viacep
